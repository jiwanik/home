Definitions

File
A logical collection of information stored under a single name.
Information
Words, numbers, pictures, and sounds that have meaning.
Directory
A logical collection of files stored under a single name.
CPU
The central processing unit. “The brain of the computer”.
Primary Storage
Temporary, volatile, (relatively) expensive, storage. Usually referring to RAM (random access mode) and is the only type of storage the CPU can access directly.
Secondary Storage
Non-volatile, (relatively) permanent, cheap and slow. Examples include: optical disc (CD, DVD), magnetic tape, hard drive (hard disc inside a drive), and flash memory (USB, phone memory).
Joint Photographic Experts Group (JPEG)
A format for compressing image files.*
Graphics Interchange Format (GIF)
A lossless format for image files that supports both animated and static images.*
Portable Document Format (PDF)
a file format that provides an electronic image of text or text and graphics that looks like a printed document and can be viewed, printed, and electronically transmitted.*
Portable Network Graphics (PNG)
a compressed raster graphic format. It is commonly used on the Web and is also a popular choice for application graphics.
